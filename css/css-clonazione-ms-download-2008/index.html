Panoramica

Internet Explorer 7 è progettato per semplificare le attività quotidiane, offrire una protezione dinamica, nonché migliorare la piattaforma di sviluppo e la gestibilità. I miglioramenti per l'utente finale includono un'interfaccia ancora più intuitiva, esplorazione a schede, avanzate funzionalità di stampa, funzionalità di ricerca più potenti, feed immediati (RSS), protezione dinamica e molto altro ancora.

Visitare il sito di Internet Explorer per ulteriori informazioni su Internet Explorer 7.

Questo download è adatto solo ai sistemi che eseguono Microsoft Windows XP Service Pack 2. Per scaricare la versione di Internet Explorer 7 per le altre versioni supportate di Microsoft Windows, vedere i Collegamenti correlati.
